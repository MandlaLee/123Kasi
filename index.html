How uploading works

For the GitHub Pages MVP, you upload media by creating a folder inside feed/. GitHub Actions scans the folder and updates data/feed.json.

feed/
  my-new-album/
    image-01.jpg
    image-02.png
    clip-01.mp4

Only upload content you own or have permission to use. For public uploads later, 123Kasi will need a backend and moderation system.
